通过这个 resp， 目前我们比较关心的字段，首先注意 changeList 是个数字，可能会有多个仓库任务：

1. changes
   1. id: taskid
   2. creator：开发者
   3. title：仓库名称
   4. manifest：代码变更详情
      1. codeElement：变更的代码详情（每个信息都特别重要）
2. diffCount
3. commentCount

## 查询任务列表

url：https://bits.bytedance.net/api/v1/dev/task/list
method：post
body：

```json
{
  "conditionGroup": {
    "conditions": [
      {
        "fieldKey": "spaceId",
        "operator": "in",
        "vals": ["201141148930"],
        "junction": "and"
      }
    ],
    "junction": "and"
  },
  "fieldKeys": [
    "title",
    "releaseTicket",
    "developers",
    "pipelineSummaryStatus",
    "project",
    "state",
    "stage",
    "createdAt",
    "creator",
    "codebaseRepo",
    "sourceBranch",
    "targetBranch",
    "teamFlow"
  ],
  "pageSize": 10,
  "page": 1,
  "aboutUsers": ["maifeng"]
}
```

1. spaceId：空间 id
2. devopsUser：开发者
3. pageSize：分页大小
